# mp3alarmClock
# S. Kleineberg / 2026                                  rev. (2025)1031 
***********************************************************************
The mp3alarmClock is an Arduino-Uno-based alarm clock that wakes you
up by playing 100 of your favourite MP3s stored on a Mico-SD-card.
The SD-card-handling and MP3-decoding and amplification is done using
an Adafruit-MP3-shield. The actual time is displayed on green 7-Segment-
LED-displays. The clock could be synchronized by a quarz crystal or
50Hz-power-frequency.
Details for the HW could be found at:
http://www.smallhill.de/html/hardware.html
***********************************************************************
